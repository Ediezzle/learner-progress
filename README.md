# Learner Progress Dashboard - Coding Challenge

## Getting Started

1. Run `composer install`
2. Configure your `.env` file from the example
3. Set up database configuration values
4. Generate the App Key: `php artisan key:generate`
5. Run migrations and seeders: `php artisan migrate --seed`
6. Start the development server: `php artisan serve`
7. Install Node dependencies: `npm install`
8. In a separate terminal, run the Vite dev server: `npm run dev`
**Note:** Both the PHP server (step 5) and Vite dev server (step 7) need to run simultaneously in separate terminal windows.
